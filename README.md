# Azure-Data-Engineering-End to End
Project Overview (Short README Version)

This project is an end-to-end Azure Data Engineering pipeline built to solve a simulated business problem and strengthen practical understanding of data pipelining concepts.

The solution ingests customer and sales data from an on-premises SQL Server into Azure Data Lake using Azure Data Factory, transforms and enriches the data using Azure Databricks following a Bronze‚ÄìSilver‚ÄìGold architecture, and serves analytics-ready data through Azure Synapse Analytics. A Power BI dashboard is used to visualize key business insights such as gender distribution, product category sales, and revenue trends, with interactive filtering by date, gender, and product category.

The pipeline is fully automated, secure, and designed to be reproducible, demonstrating real-world data engineering practices including orchestration, transformation, analytics, and reporting.
